# vagrant-alias zsh plugin

Just a bunch of aliases to speed up working with Vagrant

```
vg='vagrant status'
vgb='vagrant box list'
vgbu='vagrant box update'
vgd='vagrant destroy'
vgh='vagrant halt'
vgi='vagrant init'
vgp='vagrant provision'
vgr='vagrant resume'
vgs='vagrant suspend'
vgsh='vagrant ssh'
vgu='vagrant up'
```
